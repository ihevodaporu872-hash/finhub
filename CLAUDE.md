# Правила для Claude

## Общение
- Общайся с пользователем на русском языке
- В общении с пользователем не писать код, только общую архитектуру и необходимость внесения изменений в БД, пиши кратко

## Код
- Строгое правило — файлы не больше 600 строк. Разбивай на компоненты

## Запреты
- Запрещён локальный автоматический запуск приложений без явного разрешения пользователя
- Не делать самостоятельно коммиты в GitHub — только по явному запросу пользователя
- Не запускать самостоятельно портал для теста

## Миграции
- Все миграции сохранять в папке `sql/` для запуска пользователем на сайте Supabase в SQL Editor
